## 👨‍🏫 研究基础介绍 (Research Summary)
[cite_start]申请人长期从事电力系统信息感知相关工作。于2022年12月在东华大学信息科学技术学院获得工学博士学位，导师为欧洲科学院院士 Zidong Wang 教授 [cite: 11]。

[cite_start]本人针对非常规测量下（如测量离群值、传感器饱和、随机时延、网络攻击等）的电力系统动态状态估计问题取得了一系列研究成果 [cite: 11, 17][cite_start]。目前以第一作者身份发表 SCI 学术论文 9 篇，包括控制领域顶级刊物 **Automatica** 3 篇，以及 **IEEE Transactions on Smart Grid**、**IEEE Transactions on Industrial Informatics** 等顶级刊物 [cite: 11][cite_start]。博士学位论文被评为上海市自动化学会2022年度最佳博士学位论文 [cite: 11]。

[cite_start]本人与国内外学者保持密切合作，曾受国家留学基金委资助赴英国布鲁奈尔大学联合培养 [cite: 11]。
